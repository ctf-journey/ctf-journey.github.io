Contribute

Interested in contributing to CTF Journey?

  • CTF Journey only accepts contributions to guides
  • All contributors will be featured on GitHub

Steps for new contributors

  • Clone the respository https://github.com/ctf-journey/guide
  • Modify the category markdown file you are interested in
  • Ensure content is easy to understand and not too complex
  • Adhere to the current guides template
  • Name the commit message as “[concept/tool] added for [category]”
  • Create a pull request and wait for it to be reviewed (less than a week)

Collaboration

  • Reach out over email at privacy_mini@pm.me
